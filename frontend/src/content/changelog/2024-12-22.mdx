### New Features
- Added ability to import and display project engagement end dates from Everhour time tracking system.
- Launched new [Due on Projects](/operational-summaries/due-on-projects) dashboard that organizes projects by their engagement end dates, helping track when client contracts conclude.

### Improvements

- Enhanced [Cases](/about-us/cases) page to show engagement end dates for each project.
- Updated [Clients](/about-us/clients) page to display when project engagements are scheduled to conclude.

### Bug Fixes

- Fixed filtering on [Cases](/about-us/cases) page.
